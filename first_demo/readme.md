# 项目具体内容:
    1.首先 vs code里面的Live Sever插件要想打开的不是Safari，要改成
        "liveServer.settings.CustomBrowser": "chrome"
    2.其次 html 跟 JS 要安装的插件有:
        Live Server:            实时在浏览器打开并热重载 HTML。
        Live Preview:           Microsoft 官方的内嵌预览（可选，和 Live Server 二选一或配合使用）。
        HTML CSS Support / IntelliSense:    提供 HTML/CSS 智能提示。
        IntelliSense for CSS class names in HTML:   从项目 CSS 中补全 class 名称。
        Auto Close Tag:         自动闭合 HTML 标签。
        Auto Rename Tag:        同步重命名开始/结束标签。
        Path Intellisense:      文件路径补全（引用脚本、图片时很方便）。
    3.知识内容:
        基础语法:var , if , for , xxxx(function(){}) 等等
        获取元素，用id(#) ,class(.) 获取选择器 ,querySelector('#xxx .xxxx') // 空格代表下一级
        修改文本内容 xx.innerHTML = ''
        样式处理(css) 
        文本处理:例如加粗 : xxx.innerHTML = '<span class="bold-text"> 加粗的文本 </span>';
